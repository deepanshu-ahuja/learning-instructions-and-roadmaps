ReactJS Interview & Performance Mastery — Roadmap_React
──────────────────────────────────────────────

Goal:
Teach ReactJS from fundamentals to advanced performance and frontend optimization,
with a strong focus on:
• React mental models
• Hooks and state management
• Performance pitfalls and anti-patterns
• Frontend performance (bundle, network, rendering)
• Architecture and system design
• Redux, routing, class components (interview-focused)
• React 19 latest features

Assumption:
JavaScript fundamentals are handled in the separate JS mastery project.
